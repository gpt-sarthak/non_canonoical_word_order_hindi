Non-Canonical Word Order in Hindi

Replication of Ranjan & van Schijndel (2024) — Full Pipeline Analysis

1. Data pipeline

Sentence filtering funnel

From 13,306 raw sentences, structural filters retain 2,828 (21.2%). Dropped sentences fail projectivity, lack subject/object as direct root dependents, contain negative markers, or have fewer than 2 preverbal phrases.

Sentence length distribution (raw corpus)

Distribution of token counts across all 13,306 sentences before filtering.

2. Variant generation

Construction type distribution

DOSV (direct object fronted) and IOSV (indirect object fronted) are the non-canonical constructions of interest. SOV is canonical Hindi word order.

Variants per reference sentence

Sentences with more preverbal phrases generate more variants. Capped at 99; random sampling applied when over the cutoff. Deprel adjacency filter removes ungrammatical permutations.

3. Feature distributions (reference − variant)

Each delta = feature(reference) − feature(variant). Positive values mean the reference has higher feature value. The key question: do reference sentences consistently show lower surprisal and shorter dependencies than their variants?

Mean delta per feature per construction

If a feature predicts reference preference, its mean delta should be positive for non-canonical constructions. Surprisal features dominate; DL shows a weaker but consistent positive effect for DOSV.

4. Information status (givenness)

IS score distribution (reference sentences)

−1 = new-before-given (marked), 0 = both same givenness, +1 = given-before-new (canonical discourse order).

IS score findings

27% of reference sentences follow given-before-new order (+1) — matching the paper's prediction that reference sentences are discourse-coherent.
14% show new-before-given (−1) — these are the harder-to-process marked constructions the paper studies.
IS score is significant for DOSV (p < 0.001) — replicates the paper's finding that direct-object fronting is driven by givenness.

7. Comparison with paper results

DL is significant for both DOSV and IOSV in this replication (paper: only DOSV). The likely cause is the absence of PCFG surprisal — the paper shows PCFG absorbs DL's effect for IOSV sentences.
All coefficient directions match the paper. Surprisal features dominate. IS score is significant for DOSV. Core findings replicated.
