# non_canonoical_word_order_hindi# Non-Canonical Word Order in Hindi – Strict Replication

This project strictly reproduces:

Ranjan & van Schijndel (2024)
"Does Dependency Locality Predict Non-canonical Word Order in Hindi?"

## Components

- UD Hindi HDTB treebank
- Variant generation
- Dependency length computation
- Information status feature
- Trigram language model
- 2-layer LSTM language model
- Adaptive surprisal computation
- PCFG surprisal (5-fold CV)
- Pairwise ranking logistic regression
- McNemar statistical testing

## Structure

Modular architecture with AWS portability.
